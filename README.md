# 📅 ScheduleSync – Timetable to Calendar Automation

🚀 Turn your college timetable into a fully synced calendar in seconds.

🔗 Live Demo: https://schedulesyncc.netlify.app/

---

## 📌 Problem

Students spend a lot of time manually entering their class schedules into calendar apps.  
This process is:
- Time-consuming  
- Error-prone  
- Repetitive every semester  

---

## 💡 Solution

**ScheduleSync** automates the entire process.

📸 Upload your timetable image → 📅 Get events instantly added to your calendar  

No manual input. No mistakes. Just seamless scheduling.

---

## ⚙️ How It Works

1. Upload timetable image  
2. Extract text using OCR  
3. Convert data into structured schedule  
4. Sync events with calendar  

---

## 🧠 Features

- 📸 Image-based timetable upload  
- 🔍 Text extraction (OCR)  
- 📅 Automatic schedule generation  
- 🔗 Calendar integration  
- ⚡ Fast and user-friendly UI
